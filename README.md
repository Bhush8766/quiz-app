# 🎯 Quiz Master App

An advanced and modern Quiz Application built using React + Vite with multiple categories, dark/light theme, leaderboard, analytics, and responsive UI.

---

# 🚀 Features

## ✅ Quiz Features
- Multiple Categories
- Random Questions
- Show Correct Answers
- Score System
- Responsive Design

## 🌙 Theme
- Dark Mode
- Light Mode
- Theme saved using LocalStorage

## 📊 Analytics
- Correct Answers
- Wrong Answers
- Accuracy Display

## 🏆 Leaderboard
- Username Entry
- High Scores
- Rank System

## 🎮 Extra Features
- Progress Bar
- Beautiful UI
- Mobile Responsive
- Category Selection Dropdown
